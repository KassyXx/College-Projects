# University Projects - Systems Engineering (UPT)

Acest repository contine o selectie de proiecte dezvoltate in timpul facultatii (Anul 1 si 2) la Universitatea Politehnica Timisoara.

## Sports Field Booking System (C#)
- **Descriere:** Aplicatie de gestiune pentru baze sportive, utilizand programarea orientata obiect.
- **Concepte:** Mostenire, Polimorfism, Persistenta datelor (File I/O).

## Dynamic Programming - Triangle Path (C)
- **Descriere:** Algoritm de optimizare pentru găsirea sumei maxime într-un triunghi numeric.
- **Tehnică:** Bottom-up dynamic programming.

## Zig-Zag Matrix Generator (C)
- **Descriere:** Algoritm pentru popularea unei matrice de dimensiune 2^N intr-un model "snake/zig-zag".
- **Concepte:** Matrice, Puteri ale lui 2, Mapare coordonate.

## Data & Time Logic System (C)
- **Descriere:** Calculator pentru numarul de weekend-uri intre doua date calendaristice.
- **Detalii:** Gestioneaza ani bisecti si constrangeri de timp Unix.
